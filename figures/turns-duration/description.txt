Agent Duration Distribution Across Qwen3-Coder-30B Benchmark and Agent Runs

This figure compares per-job agent durations for six selected runs under `results/qwen3-coder-30b`, covering three benchmarks and two agent implementations. The box-and-jitter layout mirrors `figures/turns-scatter`, but replaces turn count with agent runtime in seconds so differences in how long jobs keep an agent active are easy to compare side by side.
